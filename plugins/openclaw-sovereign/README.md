# OpenClaw Sovereign Orchestration Plugin

This plugin integrates OpenClaw agents into the Sovereign Agentic OS.

## Features
- **command-logger**: Logs tool calls to the ALIGN Ledger with SHA-256 hashes.
- **boot-md**: Injects Sovereign OS ALIGN governance rules into agent system prompts.
- **tool-validation**: Validates execution requests against `openclaw_strategies.yaml`.
